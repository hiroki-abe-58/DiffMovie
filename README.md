# DiffMovie

動画メタデータ比較ツール - 2つの動画ファイルの詳細情報を比較し、変換に必要な差分を可視化します。

## 機能

- 2つの動画をドラッグ&ドロップで簡単比較
- 詳細なメタデータの差分表示（異なる項目をハイライト）
- 変換サマリーをワンクリックでコピー

### 取得する動画情報

**基本情報**
- ファイル名、ファイルサイズ、コンテナフォーマット
- 総尺、総ビットレート、ストリーム数

**映像ストリーム**
- コーデック名/詳細、プロファイル、レベル
- 解像度（幅x高さ）、アスペクト比（DAR/SAR）
- フレームレート（fps）、映像ビットレート
- ピクセルフォーマット、カラースペース、色域
- ガンマ/転送特性、カラーレンジ、HDR形式、ビット深度

**音声ストリーム**
- コーデック名/詳細、プロファイル
- サンプルレート、チャンネル数/レイアウト
- 音声ビットレート、ビット深度、サンプルフォーマット

## 必要環境

- Python 3.10以上
- ffmpeg（ffprobe）

### ffmpegのインストール（macOS）

```bash
brew install ffmpeg
```

## インストール

```bash
# リポジトリをクローン
git clone git@github.com:hiroki-abe-58/DiffMovie.git
cd DiffMovie

# 仮想環境を作成
python3 -m venv venv
source venv/bin/activate

# 依存パッケージをインストール
pip install -r requirements.txt
```

## 使い方

### 方法1: run.commandをダブルクリック（推奨）

`run.command` をダブルクリックするだけで、アプリが起動しChromeで自動的に開きます。

### 方法2: コマンドラインから起動

```bash
source venv/bin/activate
python app.py
```

ブラウザで http://127.0.0.1:7860 にアクセス

## スクリーンショット

1. 2つの動画をドラッグ&ドロップ
2. 「比較する」ボタンをクリック
3. 差分テーブルで違いを確認
4. 変換サマリーをコピーして活用

## 技術スタック

- **UI**: Gradio 6（ダークテーマ + ネオンイエロー）
